This is a simple Command line interface based game. 
The user has to input Choice of snake, water or gun by pressing 1, 2 or 3.
This program utilizes random module for the computer for its between snake water or gun.
The rules are the general accepted rules for the game which states snake drinks the water, so snake wins.
Gun kills the snake so gun wins.
Water destroys the gun so water wins.
The game runs on a while loop but if the user inputs 8 the loop breaks and game is stopped.

This is a simple and fun way to play the game.
